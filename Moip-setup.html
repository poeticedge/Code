Media over IP (MoIP) Switch Setup

Thank you for purchasing Binary’s B-900 Series Media over IP System. These network switch setup instructions apply to Araknis 210 PoE & 310 PoE switches. Refer to the MoIP Network Guide for minimum switch requirements for deployment on other network switches.

Apply the following settings to all managed swithces on the network.

Prior to connecting and powering on any MoIP devices, follow the steps below to properly configure an Araknis 210 POE / 310 POE Series switch.

  1. Factory default the Araknis switch. Press and hold the reset button for 10—15 seconds until the Status LED flashes once. This will reboot the switch and reset to factory settings.
  2. Access the local UI of the switch via Remote Access in OvrC or connect locally by directing a web browser to the IP address of the switch
    • a) Default IP Address: 192.168.20.254
    • b) Default login/password: araknis/araknis
  3. Choose ADVANCED > MULTICAST > IGMP SNOOPING and make the follow changes:
    • a) Version: V2
    • b) Report Suppression: Enabled
    • c) Unregistered IPMC Forward Action: Flood
    • d) IGMP Snooping Status: Enabled
    • e) Fast Leave: Enabled
    • f) Querier State: Enabled
    • g) Querier Version: V2
    • h) Router Settings > Router Ports Auto-Learned: Enabled
    Click Apply to save these settings
  4. Choose SETTINGS > PORTS, then verify Jumbo Frame is set to greater than 8,000 Bytes.

To complete the setup of the MoIP system, please refer to the MoIP Installation & Setup Guide
