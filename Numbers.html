Live Demo - Mobile App Setup

  1. Download the LUMA app for your device through the iTunes Store or Android Market.

  2. To add your device, open the app and press the home button on the upper left corner. Select Devices, then press the + icon to add a new device.

  3. NVR address is: lumanvrdemo.wirepathdns.com - DVR address is: lumadvrdemo.wirepathdns.com - Port : 8000 - User Name: user - Password: 1234

  4. Press "Add" button on the upper right corner to confirm and finish. Once the information is confirmed, select “Start Live View”.

Live Demo - Desktop Setup

  1. Open a supported browser and click here for NVRs, or here for DVRs.

  2. Install any add-ons or plug-ins if prompted, and follow the directions to complete the installation.

  3. If prompted to run our “Luma™ Web Video ActiveX” add-on, please enable it to always run for wirepathdns.com.

  4. Log into the device with the following credentials: Username: user - Password: 1234
