Network Diagnostic Utilities

The customer dashboard identifies the total number of monitored devices and those that need an update, have limited connectivity, or are in a critical state.

Learn more

  • Troubleshooting any issues or concerns begins in the Network Health section at the bottom of the page, where current and historical data can be found for:

    • LAN latency – indicator for an issue within the local network
    • WAN latency – consistency and quality of ISP service
    • Download and Upload Speed – Ping IP addresses on the network to see device response times and packet loss on the local network.

    Additionally, take advantage of the CRM lite functionality to organize customer data and contact information, a Google Map of the location, and secured vault for linked documentation: designs, spec sheets, bills of materials, etc.

Scan & Monitor Devices

Locate all IP devices on the network and take advantage of easily accessible controls such as reboot and update options. Enable device monitoring for any applicable device to receive notifications regarding changes in its online/offline status. Enter a unique device name for each device, configure the necessary settings - dependent upon its level of OvrC integration – and add any pertinent notes for future reference.

Learn more

    • Scan VLANS, Ports and Services - Scan the network or across multiple VLANs for available ports and their associated services: HTTP, HTTPS, telnet, SSH, and any other custom-defined combination.
    • General Device & SNMP Information - View the device's name, MAC & IP addresses, manufacturer, first and last seen time stamps, uptime, and list of any network interfaces with their respective connection speeds, states, and in/out data statistics.
    • Device Monitoring & Alerts - Choose devices to track and get automatic alerts via email or push notification for any changes in their connectivity or online status.

Interoperability

Support for third-party, open-source, and custom integrations provides detailed device information, configuration options, status updates, and controls.

Learn more

  • ONVIF Information – Camera protocol for Partners to access video streams, snapshots, user lists, firmware versions, and soft reboot for select devices. (Capabilities may vary based on manufacturer)

    PJLink Information – Standard protocol for operating compliant LAN-capable projectors: Controls include input selection, volume control, and reboot. View status information for errors (lamp error, power, etc.), and check lamp status and hours.

    Deep Integrations – Working with third-party brands allows us to offer more device data, controls, and diagnostics, although integration depth varies by device type and manufacturer.

    Complete Integration – Our proprietary brands feature simplified configuration and deployment processes, including auto-discovery, network configuration, "Apply To All" settings, automatic programming, and automated notifications.

    • Luma X20 Surveillance Systems
    • MoIP 960 AV Distribution
    • Wi-Fi Management for Araknis, Access Networks and Pakedge access points
    • Luma Insights for Control4
    • Control4 account setup and transfer

Advanced Remote Access

Web Connect allows technicians to remotely connect to a device's local user interface (UI) webpage as if they were directly connected to the customer's local network.

Learn more

    • Uses the existing, secure, authenticated communication via OvrC Pro Agent to OvrC servers.
    • Supports HTTPS (port 443), SSH, and telnet connections without needing VPNs, port forward, or complicated setups.
    • Enables cloud tools like Ping, Traceroute, and a terminal emulator for telnet and SSH to function as if they're connected locally.
